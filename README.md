## Requirements

### Running

- macOS 10.10+
- Windows 7+

### Building

- Node.js v12.14.1
- yarn v1.17.3
- electron v6.0.0

```
git clone https://github.com/emtry/StocksBar.git
cd StocksBar
npm install -g yarn
yarn install

# run
yarn start

# pack
yarn dist
```
