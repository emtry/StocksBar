## Requirements

### Running

- macOS 10.11+
- Windows 7+  (deprecated)

### Building

- Node.js v16.14.2
- yarn v1.22.18
- electron v17.3.1

```
git clone https://github.com/emtry/StocksBar.git
cd StocksBar
npm install -g yarn cross-env
yarn install

# run
yarn start

# pack
yarn dist
```
